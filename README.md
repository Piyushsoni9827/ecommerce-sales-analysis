# E-commerce Sales Data Analysis

This project analyzes a retail superstore dataset using Python to uncover sales trends and business insights.

## Tools Used
- Python (Pandas, Matplotlib, Seaborn)
- Jupyter Notebook

## Key Insights
- Technology category generated highest revenue
- Western region contributed most sales
- High discounts reduce profitability
- Seasonal trends observed in year-end months

## Project Type
Exploratory Data Analysis (EDA)
